# CODEX_COMMAND_CENTER_SETUP_GUIDE.md

This repository was initialized from the revised Phase 0 Command Center setup guide approved by Mehmetali.

## Setup Boundary

Phase 0 creates the Command Center skeleton only:

- folders
- Markdown scaffolds
- placeholder files
- setup-safe tests
- governance, learning, audit, proposal, and literature scaffolds

## Not Included In Phase 0

- scientific model code
- data download
- training or evaluation code
- raw data changes
- processed data
- experiment results
- final TUBITAK proposal language
- invented citations
