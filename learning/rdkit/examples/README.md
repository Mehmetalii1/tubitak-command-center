# RDKit Examples

## NMReDATA learning sample provenance

Dosya:
`learning/rdkit/examples/nmrshiftdb2rawdata.nmredata.sd`

Kabul edilen statü: `learning sample / pilot example`.

Bu dosya RDKit/NMReDATA öğrenimi için gerçekçi ama kontrollü bir örnek olarak
tutulur. Final TUBITAK dataset'i, source selection kararı, model-ready subset
count, benchmark dataset veya temiz bilimsel dataset değildir.

Dosya düzeyi gözlemler:

- Yaklaşık dosya boyutu: `1,164,887` byte.
- Yaklaşık satır sayısı: `37,140`.
- Opus 4.8 review raporunda yaklaşık gözlem olarak `196 kayıt` bilgisi
  raporlanmıştır.

Bu gözlemler final dataset sayımı, model-ready kayıt sayısı, lisans sonucu veya
bilimsel veri uygunluğu iddiası değildir.

Kaynak URL / erişim tarihi henüz repo içinde kayıtlı değildir. Bu bilgi
uydurulmayacak; yalnızca gerçek provenance kanıtı bulunduğunda güncellenecektir.
