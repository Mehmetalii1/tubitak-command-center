# STOP_DOING_LIST.md

## Stop Doing During Phase 1A

Phase 0 setup is closed, but its no-go boundaries still apply unless Mehmetali explicitly approves a later scientific task.

- Treating structure creation as scientific progress.
- Expanding scope because a template exists.
- Writing final proposal language before evidence exists.
- Generating citations without source verification.
- Creating dashboards or automations before they are needed.
- Starting model, data, train, or evaluate work without approval.

## Reminder

Progress must improve correctness, understanding, reproducibility, evidence quality, proposal readiness, auditability, execution clarity, or Codex discipline.
