# Data Source Decision - Evidence and Discussion

This folder holds the detailed evidence and discussion material relocated from
`docs/adr/0001-data-source.md` so the ADR can stay at template shape.

The ADR (`docs/adr/0001-data-source.md`) remains the decision record and is now
`Accepted`: NMRShiftDB2 is conditionally accepted as the first active data source.

The notes in this folder are mostly historical / pre-decision evidence and
discussion records. They do not authorize data download, parser/loader work,
processed data creation, notebook creation, split implementation, modeling,
evaluation, benchmark, score reporting, or run logging.

## Files

- `candidate-research.md` - candidate shortlist, focused verification, and the NP-MRD vs NMRShiftDB2 decision-discussion comparison.
- `verification-and-evidence.md` - remaining verification checklist, must-answer evidence collection plan, per-candidate evidence notes, feasibility note, and compact readiness summary.
- `advisor-discussion.md` - advisor/owner discussion agenda, questions, and the blank discussion outcome template.
- `DECISION_NOTE.md` - records the NMRShiftDB2 outcome while preserving the pre-decision framing of the license and narrative/scope trade-offs.

These are working evidence notes, not final proposal or literature text, and they do
not start the scientific pipeline.
