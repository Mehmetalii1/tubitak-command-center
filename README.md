# tubitak-command-center

Codex-based Command Center for the TUBITAK 2209-A project:

**Predicting functional groups from 13C NMR spectra**

This repository manages:

- project memory
- task discipline
- scientific decision records
- experiment tracking scaffolds
- learning gates
- literature evidence
- TUBITAK proposal scaffolding
- Codex session protocol
- audits and governance

Phase 0 setup is closed. The current phase is Phase 1A; see `STATE.md`.

Phase 1A still does not implement the scientific model or scientific pipeline.
